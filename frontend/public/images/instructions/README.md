# Instruction Images

This directory contains images used in the Cashly instruction modal.

## Current Images:

- `category.png` - Custom categories feature
- `transaction.png` - Transaction tracking
- `budget.png` - Budget management
- `reminder.png` - Reminder system
- `Family Wallets.png` - Family wallet sharing
- `dashboard.png` - Dashboard overview

## Image Specifications:

- **Format**: PNG or JPG
- **Aspect Ratio**: 16:9 (recommended)
- **Resolution**: 800x450px (recommended)
- **File Size**: Keep under 500KB for optimal loading

## Notes:

- Images are optional - the modal will adapt if images are not provided
- Update the `instructionData.ts` file to modify image paths
- Images should showcase the relevant features of Cashly
- The modal includes error handling for missing images
